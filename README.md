# ruby-search-sandbox
Sandbox for the https://github.com/Irostovsky/ruby-search


# Test

1. ```git clone git@github.com:Irostovsky/ruby-search-sandbox.git```
2. ```bundle install```
3. To index file ```index page1.txt```
4. To search:
  * ``` search hello word ```
  * ``` search hello AND word AND more```
  * ``` search "hello word" more```
